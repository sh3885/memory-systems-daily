# Handoffs

Create one file per completed or blocked task: `<task-id>.md`.

```markdown
Task: T-000
Status: ready_for_review | blocked | done
Owner: agent-name
Files changed:
- path/to/file
Decisions:
- decision and reason
Sources and assumptions:
- source or assumption
Validation:
- command: result
Risks:
- remaining risk or none
Next action:
- one concrete action
```

Handoffs are append-only evidence. Correct an old handoff with a new dated note instead of silently rewriting its result.
