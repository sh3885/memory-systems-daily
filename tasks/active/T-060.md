# Objective

Make the draft handler reconcile final persisted state before surfacing a
version conflict to the user.

# Result

The handler now reloads the lesson and content hash after any final version
conflict. When the revision and draft-ready state already persisted, it returns
a successful unchanged-draft response instead of a false error.
